# homebrew-ccjanus

Homebrew tap for [ccjanus](https://github.com/miya10kei/ccjanus) - Claude Code PreToolUse Hook for auto-approving bash commands.

## Installation

```bash
brew install miya10kei/ccjanus/ccjanus
```

## Update

```bash
brew upgrade ccjanus
```

## Supported Platforms

| OS | Architecture |
|----|-------------|
| macOS | Intel (x86_64) |
| macOS | Apple Silicon (aarch64) |
| Linux | x86_64 |
| Linux | aarch64 |
